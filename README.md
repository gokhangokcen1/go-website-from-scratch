Staj süresince Go/Fiber/GORM ile bir website yapacağım. 

## Proje: 
Alışkanlıklarımızı RPG oyunlar gibi yönetebileceğimi bir panel. 
- Görevleri tamamlamak XP verecek, yeterli XP seviye atlama sağlanacak.
- Tamamladığımız görevlerin çeşitliliği bizleri farklı `stat`larda geliştirecek. 
    - Müzik, tiyatro, yazma gibi görevleri tamamlamak `Sanatçı` kısmımızı geliştirirken,
    - Spor etkinlikleri `Dayanıklılık/Savaşçı`
    - Ders çalışmak, kitap okumak, bir proje üretmek `Zeka` kısımlarını geliştirir.
